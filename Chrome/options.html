Whitelist

Here you can input a whitelist for URLs which should not be hibernated when pressing the "send inactive tabs to sleep"-Button.
For example: if you want to whitelist "sub.example.com" and "test.example.com", you MUST put them both and can't just type "*.example.com"
You can also put something like "example.com/path/" in here, if you don't want to whitelist the whole domain.

URLs must be complete and include http/https!
One URL per line!
